# ReduxSimpleStarter  
Follows Udemy's [React with Redux tutorial](https://www.udemy.com/react-redux/learn/v4/)  

###To run  
    $ npm install  
    $ npm start  

###Specs:  
* React 0.14.3
* Redux 3.0.4
* Redux-promise 0.5.3
* Axios 0.13.1
* ES6
* Bootstrap v4
* OpenWeatherMap API
