# ReduxSimpleStarter  
Follows Udemy's [React with Redux tutorial](https://www.udemy.com/react-redux/learn/v4/)  

###To run  
    $ npm install  
    $ npm start  

###Specs:  
* React
* Redux
* Redux-promise
* Axios
* React-sparklines
* lodash
* ES6
* Bootstrap v4
* OpenWeatherMap API
